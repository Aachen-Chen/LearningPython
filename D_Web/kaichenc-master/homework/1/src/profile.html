First Seafront Fund

As a data Scientist Intern, I spent 6-month at First Seafront, where I applied convex optimization on 42 features of 100k+ records and reduced portfolio volatility by 27%. The project goal was to select equity portfolios that feature low-risk and index-neutrality. In calculation, I utilized quadratic programming by creatively transformed three-dimensional matrix of features into standard form. At the end, I developed a back-end python program integrating query, calculation and visualization, which allow fund managers to click-run the model on company Intranet.
